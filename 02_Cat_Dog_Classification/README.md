# Cat and Dog Classifier (CNN)

This model predicts whether an email is **Spam** or **Non-Spam**.

---

## Model

- **Architecture: MobileNetV2 - GlobalAveragePooling2D - Dense - Output**
- **Trained on:** Labeled Cat and Dog Dataset

### Try Model

* **Hugging Face Model:** [View Mode](https://huggingface.co/MrLeo0087/Cat-Dog-Classifier/blob/main/cnn_mobilenetv2_model.keras)

## Dataset

This dataset is from kaggle [Link](https://www.kaggle.com/datasets/anubhab098/cat-vs-dog-dataset)

---

## Input

- Image of Dog and Cat
- Convert image to array
- train model with array


---

## Output

- Probability can be cat or dog
