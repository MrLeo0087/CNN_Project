# Vehicle Damage Detection

This model predicts whether an vechicle is damage or not

---

## Model

- **Architecture: EfficientNetV2B3 - GlobalAveragePooling2D - Dense - Output**
- **Trained on:** Labeled damage and whole car image

### Try Model

* **Hugging Face Model:** [View Mode](https://huggingface.co/MrLeo0087/Cat-Dog-Classifier/blob/main/cnn_mobilenetv2_model.keras)

## Dataset

This dataset is from kaggle [Link](https://www.kaggle.com/datasets/anujms/car-damage-detection)

---

## Input

- Image of car or vehicle
- Convert image to array
- train model with array

---

## Output

- Probability can be damage or not damage
