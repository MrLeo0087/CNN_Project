# Mask Detection (CNN)

This model predicts whether an person wear mask or not

---

## Model

- **Architecture: EfficientNetV2B3 - GlobalAveragePooling2D - Dense - Output**
- **Trained on:** Labeled Cat and Dog Dataset

### Try Model

* **Hugging Face Model:** [View Mode](https://huggingface.co/MrLeo0087/Cat-Dog-Classifier/blob/main/cnn_mobilenetv2_model.keras)

## Dataset

This dataset is from kaggle [Link](https://www.kaggle.com/datasets/vijaykumar1799/face-mask-detection)

---

## Input

- Image of person face or use cv2 for live detection
- Convert image to array
- train model with array

---

## Output

- Probability can be mask, incorrect mask or no mask
